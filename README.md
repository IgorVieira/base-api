# Base API

[![Build Status](https://travis-ci.org/IgorVieira/base-api.svg?branch=master)](https://travis-ci.org/IgorVieira/base-api)



## Requirements

 - Nodejs >= v6.10.3 
 - MongoDB
 - NPM >= 5.0.1


## Usage

```
  https://github.com/IgorVieira/base-api.git
```
```
 npm install
```

```
 npm start
```

Open in localhost:3000/api/tasks

## Test
